1. Co to są pseudoklasy?

CSS przewiduje kilka ściśle określonych klas pozwalających na formatowanie niektórych elementów dokumentu są to Pseudoklasy.

2. Uwaga dotycząca dwukropka :

Szalenie istotnym elementem w tej konstrukcji jest dwukropek występujący pomiędzy selektorem a nazwą klasy.

3.Tabela opisująca pseudoklasy

A:link A:visited A:hover A:active A:focus A:lang A:first-child

4.Zastosowanie pseudoklasy "active"

  1. jako uzupełnienia definicji odnośników
  2. jako elementu poprawiającego czytelność formularzy

5.Przykłady kodów definiowania pseudoklasy

Arkusz wykorzystujący :link oraz :visited pseudoklasy w połączeniu z selektorem A. Przykładowa definicja sformatuje odnośnik za pomocą koloru czerwonego i po wybraniu taki odnośnik będzie miał kolor zielony.

A:link
{
color:red;
}
A:visited
{
color:green; }

Formatowanie wyglądu wszystkich odnośników w dokumencie XHTML/HTML.

A
{
font-size:12pt;
text-decoration:none;
}
A:link
{
color:navy;
}
A:visited
{
color:green;
}
A:hover
{
color:red;
text-decoration:underline;
}

Zastosowanie :focus

A:focus
{
color:brown;
}
